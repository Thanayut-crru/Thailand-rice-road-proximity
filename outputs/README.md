# Outputs

The repository commits compact results needed for review and verification:

- `tables/`: national and regional CSV results in calculated acres.
- `run_metadata.json`: analysis configuration and method-consistency metadata.
- `legacy_buffer_verification.json`: machine-readable full-grid archived-buffer audit result.
- `figures/road_type_sensitivity_national.png`: national summary chart.
- `figures/road_type_sensitivity_flowchart.png` and `.pdf`: workflow diagram.
- `figures/road_class_composition.png` and `.svg`: road-class composition.
- `figures/national_incremental_contribution.png` and `.svg`: stacked national
  contribution graph.
- `figures/regional_g3_heatmap.png` and `.svg`: regional G3 comparison.
- `figures/combined_map_panel.png` and `.svg`: shared-scale 2 × 3 map panel.
- `figures/raster_maps/`: 12 committed 600-DPI maps for visual review.

The workflow also generates 12 GeoTIFF rasters in `outputs/rasters/`. These
larger derived files are excluded from Git and can be regenerated by running
`run_analysis.ps1`.
